数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

有效括号组合需满足：左括号必须以正确的顺序闭合。

 

示例 1：

输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
示例 2：

输入：n = 1
输出：["()"]
 

提示：

1 <= n <= 8

## 思路

暴力组合所有情况, 循环判断是否优先

## 回溯法

仅当序列合法时继续递归, 当`左边括号<n`时, 可以添加左括号; 当`右边括号<左边括号`时 添加右括号
